# OneDoc

[![Build and Deploy](https://github.com/aarontorres0/OneDoc/actions/workflows/deploy.yml/badge.svg)](https://github.com/aarontorres0/OneDoc/actions/workflows/deploy.yml)

[OneDoc](https://aarontorres0.github.io/OneDoc/) allows users to upload multiple PDF files and merge them into a single PDF document.

## Getting Started

Follow these steps to get a development environment up and running:

1. Clone this repository:

   ```
   git clone https://github.com/aarontorres0/OneDoc.git
   ```

1. Navigate to the project directory:
   ```
   cd OneDoc
   ```
1. Install dependencies:

   ```
   npm install
   ```

1. Run the development server:

   ```
   npm run dev
   ```

## Technologies Used

- **Frontend:** React.js with Vite
- **Styling:** Tailwind CSS and DaisyUI
- **pdf-lib:** Library for manipulating PDF files
